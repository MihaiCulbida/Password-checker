# Password Checker
A simple, lightweight web app for checking the strength of your passwords, built on HTML, SCSS, and JavaScript.

# Features
- Real-time password strength detection
- Visual strength indicator with 4 color-coded bars
- Show/hide password toggle
- Tips to improve your password
- Clean UI

# How to open
Try it directly in your browser - open deployments and access the last link or go directly here https://mihaiculbida.github.io/Password-checker/
